You can:

  • Run tests to verify functionality

Code Example

// Create an entity with components
const entityId = devTools.createEntity();
devTools.addComponent(entityId, 'position', { x: 100, y: 200 });
devTools.addComponent(entityId, 'renderable', { 
  visible: true, 
  shape: 'circle',
  radius: 20
});
